文本强调

    文本强调能够使字体更显目，从而增强文字的阅读性。这个的实现是通过 ‘*’ or '_' 。
一个符号是代表斜体、二个代表加粗、三个代表加斜又加粗,麻蛋,怎么文章里面不能*使用*。

使用的时候需要注意：同一片段的开关标记必须一致，但是如果 * 或者 _ 两边都有空格, 则
会被视为星号和下划线的字面量.要使用星号和下划线字面量, 需要进行转义, 以区别于强调

列表

    列表有无序和有序，无序就用符号 -、+、*,后面接一个空格，有序直接使用数字1、2、3,数字后面接
'.'和一个空格,如果你的顺序是1、3、2,这样排序,会自动给你纠正将2改为4。

引言部分，你就知道怎么在Markdown 文件中建立一个区块引用，那会看起来像是你自己先断好行，然后
在每行的最前面加上 > ：

语法，包括标题、列表、代码区块等

其他

    Markdown是一种脚本语言，它的目标是：成为一种适用于网络的书写语言。HTML 是一种发布
的格式,Markdown是一种书写的格式。就这样，Markdown 的格式语法只涵盖纯文本可以涵盖的范围。
不在    Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。不需要额外标注这是 
HTML或是 Markdown；只要直接加标签就可以了。 比如 <img>,Markdown 引用图片无法控制图片的
大小,可以使用 HTML 中的 <img>,来控制大小。这只是一方面，使用Markdown与HTMl结合能够写出
更好看的博客。
